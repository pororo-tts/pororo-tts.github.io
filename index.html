Pororo TTS - Samples

Audio samples of the open-source library PORORO: Platform Of neuRal mOdels for natuRal language prOcessing. Pororo TTS features 11 multi-language support, Cross-lingual Voice Style Transfer, and Code-Switching functions.

Contents:

  • Multilingual TTS
  • Cross-lingual Voice Style Transfer
  • Code Switching

Notes:

  • These samples can be reproduced at https://github.com/kakaobrain/pororo.
  • Synthesis text to speech using trained model. Output audio's sample rate is 22050.
  • Samples shown here were selected based on diversity and quality.

Multilingual TTS

Samples generated by Pororo-TTS. Supports 11 language. (English, Chinese, Korean, Japanese, Jejueo, French, Dutch, German, Russian, Spanish, Finnish). It was trained using CSS10, KSS, and JSS datasets.
